Para que funcione el programa en node.js necesita descarga node.js a su ordenador. 
Despues agregar los siguientes comandos en orden para descargar librerias necesarias. 
  node init -y
  npm i --save rest-mssql-nodejs
  npm i express --save
  npm i cors --save
  npm i sqlserver --save
  
Y para finalizar correr el comando node combojs.js 

Nota. Siempre correr estos comandos en la terminal dentro de la carpeta de controladores
